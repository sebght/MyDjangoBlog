Pour faire un tag :
créer dans templates 'base_layout.html'
{% load static from staticfiles %}

<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>Articles</title>
    <link rel="stylesheet" href="{%static 'styles.css'%}">
  </head>
  <body>
    <div class="wrapper">
      {% block content %}
      {% endblock %}
    </div>
  </body>
</html>

Ensuite on doit modifier article_list.html :
{% extends 'base_layout.html' %}
{% block content%}
    <h1>Article List</h1>
    <div class="articles">
      {% for article in articles %}
      <div class="article">
        <h2><a href="#">{{article.title}}</a></h2>
        <p>{{article.snippet}}</p>
        <p>{{article.date}}</p>
      </div>
      {% endfor %}
    </div>
{% endblock %}
